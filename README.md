# BusinessBridge - Система управления заказами и поставками

## 📋 Описание проекта

BusinessBridge — это комплексная веб-платформа для управления заказами, поставками и взаимодействием между поставщиками и магазинами. Система обеспечивает автоматизацию бизнес-процессов, интеграцию с внешними сервисами и предоставляет удобный интерфейс для всех участников процесса.

## 🚀 Технический стек

### Frontend
- **React 18** с TypeScript — современный UI фреймворк
- **Redux Toolkit** — управление глобальным состоянием
- **Vite** — быстрая сборка и разработка
- **React Router** — маршрутизация с защищенными маршрутами
- **Framer Motion** — плавные анимации и переходы
- **Chart.js** — визуализация данных и аналитика
- **React Spring** — дополнительные анимации
- **Яндекс.Карты** — геолокационные возможности

### Backend
- **Node.js** с Express — серверная платформа
- **PostgreSQL** — надежная реляционная база данных
- **Sequelize ORM** — работа с базой данных
- **JWT** — аутентификация и авторизация
- **bcrypt** — хеширование паролей
- **CORS** — настройка межсайтовых запросов

### Интеграции и автоматизация
- **Telegram Bot** (Telegraf) — автоматические уведомления
- **OpenStreetMap Nominatim** — геокодирование адресов
- **PDFKit** — генерация документов

### Инфраструктура
- **Docker** — контейнеризация приложения
- **PM2** — управление процессами в продакшене
- **GitHub** — система контроля версий

## 🛠️ Установка и запуск

### Предварительные требования
- Node.js 18+
- PostgreSQL 12+
- Docker (опционально)

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd final-project
```

### 2. Настройка Backend
```bash
cd server
npm install
```

Создайте файл `.env` в папке `server`:
```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=your_database_name
DB_USER=your_username
DB_PASSWORD=your_password
JWT_SECRET=your_jwt_secret
BOT_TOKEN=your_telegram_bot_token
PORT=3000
```

### 3. Настройка базы данных
```bash
# Создание и заполнение базы данных
npm run db:setup

# Только миграции для продакшена
npm run db:setup:prod
```

### 4. Запуск Backend
```bash
# Разработка
npm run dev

# Продакшен
npm start
```

### 5. Настройка Frontend
```bash
cd client
npm install
```

### 6. Запуск Frontend
```bash
# Разработка
npm run dev

# Сборка для продакшена
npm run build
```

### 7. Запуск через Docker
```bash
# Сборка и запуск контейнера
docker build -t businessbridge .
docker run -p 3000:3000 businessbridge
```

## 🎯 Решенные технические задачи

### 1. Адаптивная карусель с автопереключением
**Проблема**: Создание карусели фотографий и новостей с автоматическим переключением и плавными переходами.

**Решение**: 
- Использование Framer Motion для плавных анимаций
- React Spring для дополнительных эффектов
- CSS Grid и Flexbox для адаптивности
- Таймеры с правильной очисткой для автопереключения

### 2. Адаптивная верстка для широкоформатных дисплеев
**Проблема**: Корректное отображение на экранах с различным разрешением.

**Решение**:
- CSS Grid с адаптивными колонками
- Медиа-запросы для разных breakpoints
- Относительные единицы измерения (rem, %, vw/vh)
- Flexbox для гибкой компоновки

### 3. Геокодирование адресов
**Проблема**: Преобразование текстовых адресов в координаты для карт.

**Решение**:
- Интеграция с OpenStreetMap Nominatim API
- Обработка ошибок и валидация данных
- Кэширование результатов для оптимизации
- Поддержка русского языка
